## Что добавлено в этом проходе

### 1. Увеличены image panels
- Grid-соотношения пересчитаны: панели занимают ~60–70% ширины spread на desktop
- Hero, calc, resolution panels — самые крупные
- Aspect-ratio 4:3 на 1024+, 16:10 на mobile

### 2. Comic-фон
- Halftone dots в body background
- Тонкие вертикальные и горизонтальные линии (comic ruling)
- Бумажная текстура на overlay panels
- Decorative слова на полях каждого экрана (opacity 0.08, повёрнуты): «170 м²», «8 млн ₽», «дом», «свой двор», «ипотека?», «расчёт», «проверка», «факты», «Синий Утёс» и др.
- На mobile слова скрыты

### 3. Новый блок «ПОКА ЕСТЬ ОКНО ВОЗМОЖНОСТЕЙ» (scene-6)
- Kicker + заголовок + подзаголовок
- Sticker grid: 170 м², от 8 млн ₽, от 47 тыс. ₽/м², семейная ипотека, дом с участком, навес/гараж, рядом с Синим Утёсом, можно посмотреть
- Текст с оговорками (без «всем одобрят», без «ипотека закончится»)
- Эмоциональный блок про дом
- **Сравнительные карточки**: Квартира / Вторичка / Готовый дом в Заповедном (3 колонки на 768+)
- CTA → квиз

### 4. Дожат квиз
- Бейдж «Теперь считаем вашу ситуацию»
- Крупнее карточка (padding 44px на desktop)
- Крупнее варианты (min-height 60px)
- Бумажная текстура на карточке
- Note: «Это не заявка в банк. Просто посмотрим варианты по вашему дому.»
- Progress bar 16px высота

### Как теперь работает квиз
1. Шаг 1: кнопка «Дальше» неактивна → пользователь выбирает вариант → кнопка становится активной
2. Шаги 2-4: появляется «Назад» (ghost), «Дальше» активен при выборе
3. Шаг 5 (контакты): «Назад» + зелёная CTA «Получить честный расчёт»
4. Валидация: имя ≥2 символов, телефон ≥10 цифр
5. Submit → POST /api/lead (сейчас 501) → thank-you screen
6. `console.log` payload в конце (через `window.zapovednyComic`)
